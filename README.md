## poemhub ![NPM version](https://img.shields.io/npm/v/poemhub.svg?style=flat)

Build your poem by your birthday

### Installation
```bash
$ npm install poemhub
```

### Example
```js
var poemhub = require('poemhub');
```

check this file: `poem.js`
